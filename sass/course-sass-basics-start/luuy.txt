cấu trúc thư mục 
abstracts : các thành phần CSS helper như biến , function ....
base : các thành phần dùng chung trang web ví dụ như : typographic rules hay reset rules
components : chứa các thành phần có thể sử dụng lại ví dụ như button,form,card...
layout: các thành phần chính của trang web : header,footer ...

khác nhau giữa function và mixins
function trả về variable còn mixin trả về CSS
